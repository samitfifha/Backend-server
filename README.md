# Backend-server


A backend Server for iOS mini project. API implementation with NodeJs, Express and MongoDb

The api uri preceed all API endpoints and the following endpoints are currently available :

* *GET*  /api/users
* *POST* /api/users
* *GET* /api/users/:id
* *PUT* /api/users/:id
* *DELETE* /api/users/:id
----------------------------
* *GET* /api/products
* *POST* /api/products
* *GET* /api/products/:id
* *PUT* /api/products/:id
* *DELETE* /api/products/:id
